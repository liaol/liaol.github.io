树莓派ssh中文乱码的解决办法

Posted on Dec 16 2018 In code Views: Disqus:

mac ssh登录树莓派， 中文显示乱码。解决办法如下：

  1. 运行raspi-config，出现的设置界面里进入Internationalisation Options，按空格选择zh_CN.UTF-8，zh_CN开头的都可以选上。
  2. vim /etc/profile， 加上export LANG=zh_CN.UTF-8
  3. source /etc/profile

这时应该可以显示中文了。

fibos兑换教程

Posted on Sep 1 2018 In code Views: Disqus:

最近fibos很火， 作为一名韭菜当然不容错过。
下面是从零开始兑换fo教程。
注：兑换FO的话注册EOS账户不是必须的，后面提现可能需要EOS账户，应该也可以直接提到交易所，我还没试。

兑换FO

  1. 注册fibos账户。开始根据官网教程生成公私钥后， curl注册账户一直失败， 然后只能一小时请求一次（提示The interval is less than one hour）。只好作罢。后来发现https://more.top/ 钱包有这个功能。该钱包内有生成fibos账户的功能。生成账户后保存好公钥，私钥，账户名
  2. 到交易所提现EOS到fibos指定账户。memo写上一步中注册到的fibos账户名。第一次可以只提一个eos，避免出问题损失大。提现到账后就可以查询fibos账户了。https://explorer.fibos.rocks/ 可以搜索到账户余额。这是账户中只有EOS。需要下一步兑换。

  3. 将账户中EOS兑换成FO。这一步我没用more.top钱包兑换，不知道有没有这个功能。
    我是直接用https://github.com/adshao/fibos-scripts 中的脚本。

    兑换完后https://explorer.fibos.rocks/ 即可查询到。

注册eos账户

注册eos账户需要eos， 也就是需要💰。
我是在https://eos-account-creator.com/ 创建的。

  1. 输出你要注册的账户名
  2. 输出公钥。（左边有生成公钥私钥的工具，一定要保存好私钥和公钥）
  3. 支付。有三种支付方式：a.虚拟货币（btc,eth等）b.信用卡 c.EOS
    我尝试从otcbtc提现到eos-account-creator.com指定的eos账户， 提示说不能提现到合约地址。 只好使用eth转账。
  4. 保存好你的私钥，公钥， 账户名

参考

  1. fibos脚本 https://github.com/adshao/fibos-scripts
  2. 查询eos交易 https://explorer.eosvibes.io/
  3. 查询fibos交易 https://explorer.fibos.rocks/

近期面试经历总结。

Posted on Aug 20 2018 In code Views: Disqus:

经过一波密集面试， 终于拿到心仪的offer， 于这周入职。
终于下定决心离开厦门，来到了帝都。其实自己都说不清为啥， 就是凭着一腔热血，趁着年轻多见见世面。
这次远程+现场面试应该有10家，大小公司都有， 感觉除了头条都能拿到offer， 就是钱的区别。
除了头条面试变态， 其他公司套路差不多，包括滴滴，百度，链家，陌陌等。一些常用技术+简单数据结构/算法。我猜主要看上家公司背景吧，非小公司出来的技术不太差一般都会要。 我刚好属于不会吹的那种， 更好的做法是提前准备好说辞， 包括工作中遇到的哪些挑战， 自己做了哪些牛逼的事。哎，在我看来平时的工作也没啥挑战， 都能轻松完成。 其实仔细想想， 还是平时总结的不够， 时间长了就忘了。换位思考下， 如果说不出遇到的挑战， 面试官可能以为你做的事情比较边缘或简单，甚至怀疑你的技术。 总之就是平时做好总结， 面试时大胆的吹出来。
这里列出一些知识点，如果掌握了这些基本上除头条外的国内互联网公司都能搞定了。 头条的话感觉需要补充各种计算机基础知识（如操作系统， 网络等）和刷一遍leecode。另外剑指offer一定要过一遍！
