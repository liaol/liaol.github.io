金门落地签一日游小记

2015-02-09
#旅行 

办好通行证就准备去金门玩,刚好是落地签也省得办入金证,趁着周末就去了.记在这里留个纪念.

1.办证 和落地签 #

先要办好通行证和签注，两周搞定，50块。由于是落地签，不需要办入台证。但是坑点是落地签需要返程的船票，个人定需要打进门的电话，用skype就可以，不过不是很方便。我是找票务公司订的往返票，加了30块。图个方便，不过一定要提前订，不然可能只剩很晚的船次。我去那天是周末，刚好遇到台胞返乡，只订到了上午11点的去程和下午4点的回程，行程比较赶，说多都是累。还有一个 落地签需要600新台币，建议在大陆先换好台币。

===============

2. 上j8 #

  • 登船

  • 到金门

金门的公交车比较少，再加上不熟，我是全程taxi的，一共花了1000新台币。路线是码头->昇恒昌免税店->全联福利中心（超市）。 #

  • 去免税店对面那条街的路上

  • 免税店对面那条街，在这边买了贡糖和牛肉干，店主阿姨很nice,打包很用心

  • 街尾的屈臣氏

全联福利中心其实就是个小超市，很小很小，但是东西挺便宜的。只收台币哦 #

  • 老干妈在大陆的价格简直是业界良心

  • 在免税店买的东西要在码头去拿

  • 侯船室的手机充电站

  • 没有墙的感觉真好

  • 下次再来

===============

3. tips #

  1. 免税店的烟酒一般是必带的，因为便宜啊。我带了一条中华和一条利群，店员问我要不要两条都带中华，错过今天再等明年呀，哈哈。去免税店剁手先要去一楼左边的服务台领个单子。
  2. 码头出站口右边的墙上的有各种攻略可以拿。
  3. 回来时不要轻信那种给钱叫你帮忙带烟酒奶粉的，因为你不知道他叫你带的东西里面还放了什么东西，不如说drug。我回来时就有人问60块帮带奶粉，被我无情拒绝了，在船上跟别人闲聊才知道可能有黑幕。总之莫贪小便宜即可。
  4. 台湾人很nice，咱们也要以礼待人哦。
  5. 没想到
