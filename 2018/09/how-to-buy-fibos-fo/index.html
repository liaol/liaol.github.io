fibos兑换教程

Posted on Sep 1 2018 In code Views: Disqus:

最近fibos很火， 作为一名韭菜当然不容错过。
下面是从零开始兑换fo教程。
注：兑换FO的话注册EOS账户不是必须的，后面提现可能需要EOS账户，应该也可以直接提到交易所，我还没试。

兑换FO

  1. 注册fibos账户。开始根据官网教程生成公私钥后， curl注册账户一直失败， 然后只能一小时请求一次（提示The interval is less than one hour）。只好作罢。后来发现https://more.top/ 钱包有这个功能。该钱包内有生成fibos账户的功能。生成账户后保存好公钥，私钥，账户名
  2. 到交易所提现EOS到fibos指定账户。memo写上一步中注册到的fibos账户名。第一次可以只提一个eos，避免出问题损失大。提现到账后就可以查询fibos账户了。https://explorer.fibos.rocks/ 可以搜索到账户余额。这是账户中只有EOS。需要下一步兑换。

  3. 将账户中EOS兑换成FO。这一步我没用more.top钱包兑换，不知道有没有这个功能。
    我是直接用https://github.com/adshao/fibos-scripts 中的脚本。

    兑换完后https://explorer.fibos.rocks/ 即可查询到。

注册eos账户

注册eos账户需要eos， 也就是需要💰。
我是在https://eos-account-creator.com/ 创建的。

  1. 输出你要注册的账户名
  2. 输出公钥。（左边有生成公钥私钥的工具，一定要保存好私钥和公钥）
  3. 支付。有三种支付方式：a.虚拟货币（btc,eth等）b.信用卡 c.EOS
    我尝试从otcbtc提现到eos-account-creator.com指定的eos账户， 提示说不能提现到合约地址。 只好使用eth转账。
  4. 保存好你的私钥，公钥， 账户名

参考

  1. fibos脚本 https://github.com/adshao/fibos-scripts
  2. 查询eos交易 https://explorer.eosvibes.io/
  3. 查询fibos交易 https://explorer.fibos.rocks/
